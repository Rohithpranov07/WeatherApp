## Setup

1. Clone the repository
2. Install dependencies: `npm install`
3. Get a free API key from [OpenWeatherMap](https://openweathermap.org/api)
4. Add your API key in `src/App.jsx` (line 289)
5. Run the dev server: `npm run dev`

## Tech Stack
- React 18
- Vite
- Tailwind CSS
- Lucide React (icons)
